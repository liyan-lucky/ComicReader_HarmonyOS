# 隐私说明

ComicReader_HarmonyOS 当前设计为本地 App 加公开网络读取工具。

## 本地数据

App 可能在本地保存以下数据：

- 搜索设置；
- 远程规则地址；
- 书架；
- 阅读历史；
- 阅读和显示偏好；
- 搜索引擎配置；
- 用户手动填写的公开 API Key 或搜索配置。

这些数据应仅用于 App 本地功能，不应提交到仓库。

## 网络访问

App 申请 `ohos.permission.INTERNET`，用于访问用户选择的公开页面、公开 API、搜索引擎和远程规则索引。

## 不应包含的行为

本仓库不应包含：

- 账号登录上传；
- 广告 SDK；
- 用户追踪；
- 遥测统计；
- 云同步；
- 后台上传用户搜索、书架或阅读历史。

如果后续添加网络同步、崩溃上报、统计分析、广告、账号登录或云服务，应在本文件中补充说明，并在 App 内提供清晰提示。
