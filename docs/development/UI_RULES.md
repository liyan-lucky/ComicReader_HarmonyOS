# UI 规范

## 全局风格

- APP 主色：`#34C759`。
- 绿色只用于选中、主按钮、强调状态和正向状态。
- 关闭、停用、禁用使用灰色。
- 页面背景使用浅灰或暗色主题背景。
- 卡片使用圆角和轻阴影，避免硬边框。

## 底部 Tab

最终 Tab：

1. 搜索
2. 书架
3. 历史
4. 设置

要求：

- 选中图标和文字为绿色。
- 不使用大面积绿色填充块。
- 关于不再单独作为 Tab。
- 书架 Tab 保留名称，但页面内容定位为热门题材推荐。

## 搜索页

搜索首页：

- 上方/中部 APP 图标。
- 中间一个搜索框。
- 搜索框上方图形直接使用 `$media:icon` PNG 资源。
- 不显示任何设置入口。
- 不显示搜索模式、来源过滤、主题、语言、演示按钮。
- 不显示复杂说明流程。

搜索结果：

- 搜索结果只提供“列表”和“网格”两种视图。
- 默认三列图片网格。
- 网格卡片固定三列，卡片只包含图片和名称。
- 列表卡片为低高度横条样式，小图片在前，名称在后。
- 列表名称最多显示两行，超出省略。
- 排列切换只在有搜索结果时显示。
- 标题显示必须经过清洗，避免重复名称、重复分隔片段和整段重复造成过长。

## 书架页 / 热门题材

- 不展示收藏内容。
- 不展示阅读历史。
- 不提供清历史、清收藏等管理功能。
- 页面标题使用“热门题材”。
- 展示自动推荐说明卡片。
- 展示题材卡片网格，例如玄幻、修真、恋爱、校园、宫斗、都市、热血、冒险、悬疑、科幻、搞笑、治愈。
- 点击题材卡片后跳转搜索并自动搜索该题材。
- 后续热门题材由规则仓库或远程目录索引自动更新。

## 历史页

包含：

- 历史记录。
- 收藏记录。
- 下载。
- 清空历史。
- 清空收藏。
- 收藏打开。
- 收藏移除。

## 设置页

设置页分区：

- 基础设置。
- 搜索设置。
- 公开源开关。
- 显示与阅读。
- 搜索引擎清单。
- 远程规则。
- 高级规则。
- 关于。

## APP 图标

- APP 桌面显示图标、启动图标、搜索首页上方图形都统一使用 PNG。
- PNG 图标来源为前期沟通预览图中已确认的图形扣图。
- 资源路径固定为 `entry/src/main/resources/base/media/icon.png`。
- `module.json5` 中 `icon` 和 `startWindowIcon` 继续引用 `$media:icon`。
- 不再使用同名 `icon.svg`，避免资源名冲突。
- 构图方向：漫画浏览器、绿色主视觉、漫画/书页/搜索元素结合。
- 可以按用户反馈继续微调。

## 图标资源

- APP 主图标使用 PNG。
- Tab、设置、历史、阅读控制图标继续使用统一 SVG 线性风格。
- Tab、设置、历史、阅读控制图标必须保持同一视觉语言。
