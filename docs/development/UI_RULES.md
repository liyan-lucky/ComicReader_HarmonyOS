# UI 规范

## 全局风格

- APP 主色：`#34C759`。
- 绿色只用于选中、主按钮、强调状态和正向状态。
- 关闭、停用、禁用使用灰色。
- 页面背景使用浅灰或暗色主题背景。
- 卡片使用圆角和轻阴影，避免硬边框。

## 底部 Tab

最终 Tab：

1. 搜索
2. 书架
3. 历史
4. 设置

要求：

- 选中图标和文字为绿色。
- 不使用大面积绿色填充块。
- 关于不再单独作为 Tab。

## 搜索页

搜索首页：

- 上方/中部 APP 图标。
- 中间一个搜索框。
- 不显示任何设置入口。
- 不显示搜索模式、来源过滤、主题、语言、演示按钮。
- 不显示复杂说明流程。

搜索结果：

- 默认三列图片网格。
- 支持列表 / 网格切换。
- 排列切换只在有搜索结果时显示。
- 结果卡片优先展示封面和来源。

## 书架页

- 书架功能保留。
- 不展示收藏内容。
- 收藏内容迁移到历史页。

## 历史页

包含：

- 历史记录。
- 收藏记录。
- 下载。

## 设置页

设置页分区：

- 基础设置。
- 搜索设置。
- 公开源开关。
- 显示与阅读。
- 搜索引擎清单。
- 远程规则。
- 高级规则。
- 关于。

## APP 图标

- 使用自绘 SVG。
- 构图方向：漫画浏览器、绿色主视觉、漫画/书页/搜索元素结合。
- 可以按用户反馈继续微调。

## 图标资源

- 使用 SVG。
- 使用统一线性图标风格。
- Tab、设置、历史、阅读控制图标必须保持同一视觉语言。
