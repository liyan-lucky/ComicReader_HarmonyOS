# 隐私说明

ComicReader_HarmonyOS 当前设计为本地 App 加公开网络读取工具。

## 本地数据

App 可能在本地保存：

- 搜索关键词；
- 阅读历史；
- 书架记录；
- 远程规则地址；
- 搜索设置和阅读设置。

这些数据默认只用于本地功能，不应上传到项目服务器。

## 网络访问

App 使用 INTERNET 权限访问：

- 用户主动输入或点击的公开网页地址；
- 搜索引擎公开结果页；
- 远程规则仓库；
- 公开漫画页面中的图片资源。

## 不应收集的内容

当前设计不应收集：

- 账号密码；
- Cookie；
- 支付信息；
- 通讯录；
- 精确定位；
- 设备唯一标识；
- 私钥或证书。

## 后续变更要求

如果后续新增统计、广告、崩溃上报、账号登录、云同步或 API Key 持久化，必须同步更新本隐私说明。
